# programming-Part2-CyberSecurityChatBot

## Project Overview
The Cybersecurity Awareness Chatbot is a desktop application developed using C# and Windows Presentation Foundation (WPF) in Visual Studio.

The chatbot helps users learn about cybersecurity topics such as:
- Password safety
- Phishing
- Privacy
- Online scams

The application uses object-oriented programming principles and includes features such as keyword recognition, sentiment detection, and memory storage

## Features
- Graphical User Interface (GUI) using WPF
- Cybersecurity awareness responses
- Keyword recognition
- Sentiment detection
- Memory functionality
- Object-oriented programming structure

## Technologies Used
- C#
- WPF (Windows Presentation Foundation)
- .NET 10.0
- Visual Studio

## Classes Used
### ChatBot.cs
Controls chatbot communication and responses.

### KeywordResponder.cs
Recognises cybersecurity keywords and generates responses.

### SentimentDetector.cs
Detects simple user emotions and sentiments.

### MemoryStore.cs
Stores user information during chatbot interaction.

### MainWindow.xaml
Designs the graphical user interface.

### MainWindow.xaml.cs
Controls GUI events and chatbot interactions

## Cybersecurity Topics Supported
- Password security

- Privacy protection
- Online scams

## How to Run the Application
1. Open the project in Visual Studio.
2. Build the solution.
3. Click Start to run the application.
4. Type messages into the textbox
